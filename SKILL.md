# daily-ai-trends

Use `scripts/generate-feeds.js` to produce normalized JSON feeds under `data/`.

## Workflow

1. Install dependencies with `npm install`
2. Run `npm run generate`
3. Publish or consume `data/feed-all.json` or source-specific feed files from downstream apps

## Files

- `scripts/generate-feeds.js`: feed generator
- `.github/workflows/generate-feeds.yml`: scheduled publishing

## Source rules

- GitHub: scrape trending repositories
- Hugging Face: fetch daily papers API
- X: consume the public `follow-builders` feed
- Reddit: parse the latest report markdown generated by `reddit-ai-trends`
- Keep this repo as a normalized aggregation layer; leave product-level fallback policy to downstream consumers
